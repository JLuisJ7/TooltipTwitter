Crearemos un plugin para Wordpress que cumpla las siguientes funciones:

1.- Al seleccionar un texto en la pagina web debe aparecer un icono de twitter
    encima del texto seleccionado
2.- Al hacer click saldra el popup para compartir en twiiter donde se compartira
    el texto seleccionado, la url del sitio y el usuario de twitter
